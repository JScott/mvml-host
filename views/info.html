Why is MVML the best choice?

You have a lot of choices to render 3D web content. However, you'll find that MVML combines the best of the competition.

Portable

MVML works with your favourite browser and webpages.

  • Renders in Chrome, Firefox, and IE
  • Built to work on mobile browsers as well

Simple

MVML exposes advanced concepts in simple ways.

  • Easy for non-technical artists to write
  • Built on easy-to-understand three.js abstractions
  • YAML-based syntax instead of bloated XML

Robust

MVML benefits from the web dev community.

  • Uses the same infrastructure as 2D web
  • Deployable with DevOps best practices
