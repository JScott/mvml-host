Why is MVML for me?

Many frameworks provide one-off 3D web experiences and proprietary 3D browsers.
MVML specializes in helping you build and share a network of first-person worlds.

Portable

  • Renders in your favourite browser
  • Built with mobile in mind

Accessible

  • Easy for non-technical artists to write
  • A consistent first-person experience

Robust

  • Built on Open Source web dev libraries
  • Exposes JS for advanced scripting

Getting Started

MVML isn't natively supported by browsers so it needs to be parsed into HTML/JS. You can utilize our API or host your own MVML parsing server to do this for you.

Quick

The easiest way to utilize this is to make an API request via JavaScript. These examples show you how to have the JavaScript do the API call for you and edit MVML in-line or in another file.

Advanced

If you have access to server-side code you can also make a call from there. These examples show how I do it for this very page in NodeJS. It should be pretty similar across frameworks.
